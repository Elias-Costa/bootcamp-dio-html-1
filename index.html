A tag < p >

A tag < p > é utilizada para criar um parágrafo. Por exemplo, em um texto com 4 parágrafos, o ideal é criar uma tag < p > para cada parágrafo do texto.

A tag < a >

A tag < a > é utilizada para criar um link, onde pode redirecionar o usuário para uma outra parte da página ou até uma página qualquer. Possui um atributo chamado target, onde pode ser passado o valor '_self' caso queira redirecionar o usuário para o link abrindo-o na mesma guia ou o valor '_blank', que abre o link em uma nova guia.

A tag < ol >

A tag < ol > é utilizada para criar uma lista ordenada, onde cada item será acompanhado de um número. A numeração é dada de forma crescente. Para adicionar um item na lista ordenada, tem que criar uma tag filha chamada < li > e dentro dela passar o conteúdo do item.

A tag < ul >

Também cria uma lista, mas neste caso, a lista não é ordenada, cada item da lista é acompanhado por um símbolo que não remete à numeração.
